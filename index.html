關於我

我目前是一名前端工程師，同時從事教學與網頁接案，主要專注於 UI 開發、互動介面實作與前端架構規劃。熟悉 HTML/CSS、RWD、設計系統、React 與現代前端工作流程。

我的背景來自於自學與技術分享，做事風格偏向「講清楚、重視溝通、喜歡把複雜的概念變得好理解」。我熱愛整理知識，也樂於協助團隊建立一致的前端標準。

  • 能協助你建立乾淨、有結構的前端 UI
  • 擅長技術教學、文件化與溝通協作
  • 可支援網站切版、RWD、互動設計與前端最佳化
