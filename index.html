關於我

您好！我目前是一名全職前端工程師，熱衷於探索 React 與 Vue 生態系的最新技術。除了日常開發，我也投入時間進行技術教學與接案工作。我擅長將複雜的設計需求轉化為像素完美（Pixel-perfect）且高效能的網頁介面。

我相信技術是為了解決問題而存在。我擁有跨領域的協作背景，這養成了我「重視溝通」與「換位思考」的工作風格。我喜歡把複雜的技術概念拆解成清晰易懂的語言，確保團隊成員與客戶都能對專案目標有一致的理解。

  • 具備現代化前端架構設計能力，注重程式碼可維護性
  • 擅長跨部門溝通，能有效橋接設計與工程思維
  • 熱愛分享與教學，能協助團隊新人快速上手
