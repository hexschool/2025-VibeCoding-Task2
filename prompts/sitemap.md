# 網頁結構

## 首頁主題（Home）

- 標題：
  簡短、抓住核心價值（例如「用木頭般的溫度，打造能陪伴你的網站」）

- 副標：
  補充定位與特色（自然、溫暖、舒服、乾淨）

- 引導語：
  強調品牌故事感 / 使用者體驗

## 服務內容（Services）

- 每項服務：
  標題：簡短、可愛 / 溫暖
  描述：說明「能為客戶帶來的感受與好處」
  適合對象 / 特點：強調使用者需求與場景

- 服務範例：
  客製化網站製作 & 前端切版
  品牌故事 / 形象網站
  活動頁 / Landing Page
  網站視覺優化 / 舊站改造

## 關於我 / About

簡短自我介紹 + 專長
設計理念與核心信念
語氣：溫暖、自然、療癒感、親近

## 作品案例 / Portfolio

每個作品包含：

- 標題 / 類型
- 簡短描述（風格、色調、重點效果）
- 客戶收益 / 使用者感受

## 聯絡頁 / Contact

簡單呼籲行動：邀請合作、一起打造溫暖網站
保持溫暖、療癒、自然感
